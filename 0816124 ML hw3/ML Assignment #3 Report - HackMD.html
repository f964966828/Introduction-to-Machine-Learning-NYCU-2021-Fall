# ML Assignment #3 Report - name: 林浩君 - student ID: 0816124 ## K-means Clustering ### 1-2. Data Input and Data Preprocessing - 從txt檔裡讀取資料存成DataFrame - 對DataFrame shuffle ![](https://i.imgur.com/12xwDS7.png) ### 3. K-means Clustering - 首先將dataset切成x(features), y(labels) - 照著pseudocode實作演算法, 並回傳 - centers: 中心點座標 - clusters: 每個點所屬的cluster index ``` def k_means(x, k): centers = np.array(x.sample(k)) clusters = np.zeros(len(x)) iterations = 100 for iteration in range(iterations): for idx, data in enumerate(x.iloc): distance = [np.linalg.norm(center - data) for center in centers] clusters[idx] = np.argmin(distance) centers = [np.array(x[clusters == idx]).mean(0) for idx in range(k)] return centers, clusters ``` ### 5. Results - 對於每個cluster, 用實際的label去統計這個cluster中的分布, 再以數量最多的當作predict label ![](https://i.imgur.com/xmVN02e.png) - 對以上case而言 - cluster 1 的預測label為1 - cluster 2 的預測label為3 - cluster 3 的預測label為2 - 以上述的預測label來產生y_pred, 得到result ![](https://i.imgur.com/78QIQac.png) ### 6. Comparison & Conclusion - k-means做出來效果比我想像中好, 我原本以為這麼簡單的演算法做不太起來 ### 7. Questions - 選擇第1, 5個feature去做k-means, 並跟上面一樣用該cluster最多的label數來當作預測label ![](https://i.imgur.com/GiR3zxT.png) ## Support Vector Machine ### 1-2.
