  • 我一開始刻沒有laplace版本的時候, 當prob.遇到0取log會跳error, 所以我第一版的作法是prob.遇到0就break, 這樣做下來準確率大概落在80%, 後來我想說log取0應該是負無限大, 直接break應該不太對, 所以改第二版成遇到0就把prob.設成一個很小的數字, 準確率就跑到96%左右, 甚至跟有laplace smoothing的差不多, 我不太確定這樣做是不是合法的, 因為laplace smoothing好像就是要避免zero prob. , 所以holdout case中有無smoothing的數據一樣我想可能是這個原因

4. Model Construction

  • construct a class with functions below:
    1. __init__(self, x, y)
      • x: features of all data
      • y: labels of all data
    2. normpdf(self, x, feature_type, class_type)
      • x: features of a data
      • calculate conditional probability by normal distribution
    3. naive_bayes(self, data, label)
      • given features of data and a specific label
      • calculate by calling normpdf()
      • It’s the Naive Bayes divier M(q)M(q) without argmax part
      • 因為 softmax 會讓有些 prob 趨近於 0, python 精度不夠時就會直接判定成 0, 所以我在公式後加上 1e-7, 免得跳 error
    4. predict(self, x)
      • x: features of all data
      • for each data, calling naive_bayes() of all labels and store argmax one as answer
      • store all data’s answer into a list called result and return
    5. cal_confusion_matrix(self, x, y):
      • calling predict() and get the result
      • calculate confusion matrix by compare result and y
      • return confusion matrix
